# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-05-21

### Added
- 初始版本发布
- 实现三种移动平均算法：
  - 简单移动平均(SMA)
  - 指数移动平均(EMA) 
  - 递归移动平均(RMA)
- 基础测试套件
- 性能基准测试

### Fixed
- 修复浮点数比较精度问题
- 解决文档测试链接问题
